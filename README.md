# NextSet - Proyecto Intermodular Final DAW (Convocatoria 2025-2026)

Este repositorio contiene el proyecto final para el ciclo de Desarrollo de Aplicaciones Web (MEDAC - DAVANTE), calificado con un **8/10**. 

- NextSet no es solo una aplicación web: es un proyecto empresarial y técnico completo (SaaS) diseñado para resolver la gestión ineficiente de entrenadores personales y gimnasios que siguen dependiendo de canales como WhatsApp o PDFs físicos. El proyecto abarca desde el desarrollo del software hasta el plan de viabilidad y despliegue del negocio.

---

## 1. Arquitectura y Stack Técnico (Plan de Desarrollo):

Para que la aplicación sea ligera, rápida en conexiones móviles y escalable, estructuré el desarrollo aplicando las competencias del ciclo.

* Frontend: HTML5, CSS3 y JavaScript. Diseñado bajo enfoque *mobile-first* para que los clientes registren marcas y series cómodamente en la sala de entrenamiento.
* Backend: Lógica de negocio y persistencia en el servidor desarrollada en PHP y Java.
* Bases de Datos: Modelado relacional y gestión para el control de perfiles de clientes, entrenamientos y contabilidad.
* Tecnologías XML: Tratamiento, estructuración y validación de datos del sistema utilizando DTD, XML Schema, XPath, XSLT y XQuery.

---

## 2. Funcionalidades del Software Implementadas:

### Vista del Entrenador / Administrador
* Gestión de Clientes: Automatización y asignación de rutinas semanales personalizadas en pocos minutos.
* Control Financiero: Panel en tiempo real para la gestión de ingresos, beneficios y suscripciones.
* Marketing: Módulo de perfil único para que entrenadores o centros promocionen su actividad mediante pago mensual.

### Vista del Cliente (Optimizado para Smartphone)
* Trackeo en tiempo real: Interfaz interactiva para marcar series terminadas, anotar pesos levantados y calcular kilocalorías diarias.

---

## 3. Plan Empresarial y Viabilidad (Full Project):

Para defender el proyecto ante el tribunal, además del código, desarrollé la estructura organizativa y financiera que justifica su viabilidad real:

* Forma Jurídica y Modelo de Negocio: Definición de los trámites de constitución de la empresa, obligaciones fiscales y calendario de tributación para un modelo SaaS.
* Gestión de Recursos Humanos: Elaboración del organigrama de la empresa, fichas técnicas de descripción de puestos de trabajo y el plan de prevención de riesgos laborales (PRL).
* Estudio Financiero: Análisis del plan de inversión inicial, búsqueda de vías de financiación/subvenciones y estudio detallado de los costes de producción para asegurar la rentabilidad.
